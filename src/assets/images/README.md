# Images Directory

This directory contains all the images used in the project.

## Structure

- `logos/` - Brand logos and icons
- `photos/` - Photography and image content
- `backgrounds/` - Background images and patterns
- `team/` - Team member photos
- `clients/` - Client logos and related images

## Guidelines

1. Use descriptive filenames in kebab-case (e.g., `team-member-jane.jpg`)
2. Optimize images before adding them
3. Include alt text when using images in components
4. Consider using WebP format for better performance
